HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard makeup language for craeting web pages and web applocations. Web browsers receive HTML documents from a web server or from local storage and render them into mutimedia web pages. HTML describes the structure od a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered pages. It provides a means to create structured documents by denoting structural semantics for text such as headings, paraghaphs, lists, links, quotes and other itens. HTML elements are delineated by tags, written using angle brackets.
